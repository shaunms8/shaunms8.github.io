# Original Artifact

This folder contains the original DAT 375 Jupyter Notebook used as the baseline artifact for the CS 499 Capstone project.

The notebook represents the Miami Crime and Storm Data Analysis System before any CS 499 enhancements were made. It was originally completed as a notebook-based data analysis project using Python, pandas, MySQL data, and visualization output.

The enhanced version of this project is stored separately and includes modular Python files, improved organization, expanded analysis, and updated reporting.
